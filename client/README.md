
## Whiteboard App

A full-stack web application that provides users with an interactive whiteboard to draw, erase, and download their sketches.
Link: https://my-whiteboard-app-project.vercel.app/

## Features

Interactive Drawing Canvas: Draw freely using mouse or touch input with smooth and responsive strokes

Erase and Undo/Redo: Easily erase parts of the drawing and navigate through previous states with undo/redo functionality

Download as PNG: Export your drawings as a PNG image file

Responsive Design: Works across desktops, tablets, and mobile devices
